PDF Form to CiviForm JSON

Convert PDF forms to CiviForm JSON. Enter a server-side directory path for multiple files or select a single PDF.

Instructions:

  1. Single File Processing:
    • To process a single PDF, select it using the file selection button.
  2. Directory Processing:
    • Enter the server directory path where your PDF files are located. Default: `~/pdf_forms`
  3. Click "Convert" (for a single file) or "Process Directory" (for multiple files on the server) to generate the CiviForm JSON output.
  4. Copy the CiviForm JSON output and import it to CiviForm.
