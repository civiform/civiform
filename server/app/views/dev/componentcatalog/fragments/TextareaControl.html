Name Required Type Details
Attributes
id required string Unique identifier for the textarea element.
label required string Visible label text for the textarea.
name required string Form field name. Supports nested names.
aria-* optional string Any aria-* attributes are passed through to the rendered textarea element.
data-* optional string Any data-* attributes are passed through to the rendered textarea element.
disabled optional boolean Set to "true" to disable the field.
help-text optional string Help text displayed below the label.
is-valid optional boolean Controls validation state. "true" or "false". Defaults to "true".
markdown-enabled optional boolean Set to "true" or "false" to enable or disable markdown support info line in the textarea.
maxlength optional integer string Maximum number of characters allowed.
minlength optional integer string Minimum number of characters allowed.
placeholder optional string Placeholder text shown when the textarea is empty.
readonly optional boolean Set to "true" to make the field read-only.
required optional boolean Set to "true" to mark the field as required.
size optional string Visual width. Allowed values: 2xs, xs, sm, small, md, medium, lg, xl, 2xl.
validation-class optional string Fully qualified Java class name used to derive client-side validation constraints.
validation-field optional string Field name within the validation class to extract Play Framework validation constraints from.
validation-message optional string Error message displayed when the field is invalid.
value optional string Initial text content of the textarea.

Basic Textarea

Java

public record BasicTextarea(
  String id,
  String name,
  String label,
  String placeholder) {

  public BasicTextarea() {
    this("comments",
      "comments",
      "Comments",
      "Enter your comments here");
  }
}

Thymeleaf

<cf:textarea
  th:id="${model.basicTextarea.id()}"
  th:name="${model.basicTextarea.name()}"
  th:label="${model.basicTextarea.label()}"
  th:placeholder="${model.basicTextarea.placeholder()}"
/>

Rendered HTML

<div class="usa-form-group">
  <label class="usa-label" for="comments">Comments</label>
  <textarea
    class="usa-textarea"
    id="comments"
    name="comments"
    placeholder="Enter your comments here"></textarea>
</div>

Java

public record TextareaWithError(
  String id,
  String name,
  String label,
  String isValid,
  String validationMessage,
  String rows) {

  public TextareaWithError() {
    this("feedback",
      "feedback",
      "Feedback",
      "false",
      "Feedback must be at least 10 characters",
      "4");
  }
}

Thymeleaf

<cf:textarea
  th:id="${model.textareaWithError.id()}"
  th:name="${model.textareaWithError.name()}"
  th:label="${model.textareaWithError.label()}"
  th:is-valid="${model.textareaWithError.isValid()}"
  th:validation-message="${model.textareaWithError.validationMessage()}"
  th:rows="${model.textareaWithError.rows()}"
  required="true"
/>

Rendered HTML

<div class="usa-form-group usa-form-group--error">
  <label class="usa-label" for="feedback">Feedback</label>
  <span id="feedback-error" class="usa-error-message" role="alert">
    Feedback must be at least 10 characters
  </span>
  <textarea
    class="usa-textarea usa-input--error"
    id="feedback"
    name="feedback"
    rows="4"
    required="true"
    aria-describedby="feedback-error"
    aria-invalid="true"></textarea>
</div>
