Name Required Type Details
Attributes
id required string Unique identifier for the radio group.
label required string Legend text for the radio group fieldset.
name required string Form field name shared by all radio buttons in the group.
aria-* optional string Any aria-* attributes on the group are passed through to rendered elements.
data-* optional string Any data-* attributes on the group are passed through to rendered elements.
disabled optional boolean Set to "true" to disable all radio buttons in the group.
help-text optional string Helper text displayed below the legend.
is-valid optional boolean Controls validation state. "true" or "false". Defaults to "true".
readonly optional boolean Set to "true" to make all radio buttons in the group read-only.
required optional boolean Set to "true" to mark the group as required.
tiled optional boolean Set to "true" or "false" to use USWDS tile styling. Defaults to "true".
validation-class optional string Fully qualified Java class name used to derive client-side validation constraints.
validation-field optional string Field name within the validation class to extract Play Framework validation constraints from.
validation-message optional string Error message displayed when the group is invalid.
Nested content (<item> attributes)
child <item> id required string Unique identifier for this individual radio input.
child <item> value required string The form value submitted when this radio button is selected.
child <item> description optional string Additional descriptive text shown below the item label.
child <item> disabled optional boolean Set to "true" to disable this individual radio button.
child <item> label optional string Display label for this radio option.
child <item> selected optional boolean Set to "true" to pre-select this option.

public record BasicRadio(
  String name,
  String label,
  ImmutableList<RadioOption> options) {

  public BasicRadio() {
    this("size",
      "Choose a size",
      ImmutableList.of(
        new RadioOption("size-small", "small", "Small"),
        new RadioOption("size-medium", "medium", "Medium"),
        new RadioOption("size-large", "large", "Large")));
  }
}

public record RadioOption(String id, String value, String label) {}

Thymeleaf

<cf:radio
  th:name="${model.basicRadio.name()}"
  th:label="${model.basicRadio.label()}"
>
  <item
    th:each="item: ${model.basicRadio.options()}"
    th:id="${item.id()}"
    th:value="${item.value()}"
    th:label="${item.label()}"
  ></item>
</cf:radio>
