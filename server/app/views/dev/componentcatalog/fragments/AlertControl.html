Alert Component

Overview

The <cf:alert> custom tag creates USWDS (U.S. Web Design System) alert components in Thymeleaf templates. It transforms simple markup into properly structured alert HTML with appropriate CSS classes and wraps content in the required usa-alert__body structure.

Attributes

Attribute Type Default Description
type String "info" Alert type: info, warning, error, success, or other USWDS alert types
slim Boolean false Creates a slim alert with reduced padding
no-icon Boolean false Removes the icon from the alert

Content Elements

Heading Elements (h1-h6)

  • Automatically styled with the usa-alert__heading class
  • Must be the first element in the alert if used
  • Any existing CSS classes on the heading are preserved
  • The heading tag itself is preserved (h1 stays h1, h2 stays h2, etc.)

<text> Element

  • Transforms into a <p class="usa-alert__text"> tag
  • Use for simple text content without complex HTML markup
  • Content between the tags is preserved as-is

<content> Element

  • Inserts rich HTML content without transformation
  • Use for complex markup like lists, multiple paragraphs, links, etc.
  • All HTML within <content> tags is rendered exactly as written
  • Does not wrap content in any additional elements

Validation Rules

The processor enforces these rules and will throw a TemplateProcessingException if violated:

1. Slim + Heading Restriction

Cannot use slim="true" with heading elements (h1-h6).

Error message: "Invalid cf:alert configuration: slim="true" cannot be used with heading elements (h1-h6)"

2. Text vs Content Exclusivity

Cannot use both <text> and <content> elements in the same alert.

Error message: "Invalid cf:alert configuration: cannot use both <text> and <content> elements. Use only one."

3. Heading Position Requirement

Heading elements (h1-h6) must be the first element in the alert, before any other content.

Error message: "Invalid cf:alert configuration: heading elements (h1-h6) must be the first element in the alert"

<cf:alert type="info">
  <h4 th:name="${variable1}">Lorem Ipsum</h4>
  <text>Lorem Ipsum</text>
</cf:alert>

Rendered HTML

<div class="usa-alert usa-alert--info">
  <div class="usa-alert__body">
    <h4 class="usa-alert__heading">Lorem Ipsum</h4>
    <p class="usa-alert__text">Lorem Ipsum</p>
  </div>
</div>
